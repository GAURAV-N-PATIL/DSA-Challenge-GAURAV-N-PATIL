### Problem

Given two sorted arrays `nums1` and `nums2`, merge them into one sorted array.
The final result must be stored inside `nums1`.

### Approach

* Use the two-pointer method.
* Compare elements from the end of both arrays.
* Place the larger element at the correct position in `nums1`.

### Complexity

* Time: O(m + n)
* Space: O(1)
